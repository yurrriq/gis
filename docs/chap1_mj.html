1.3-1 PSpace
‣ PSpace( global variable )

\((\mathbb{Z}, (\mathbb{Z},{+}), int)\) where \(int{:}\mathbb{Z}{\times}\mathbb{Z}{\to}\mathbb{Z}\) and \(int(s,t) = t-s\).
